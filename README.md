# SOC Portfolio (Splunk • Wireshark • TheHive)

This repository contains lab-based, SOC-aligned work products written in an analyst style: triage notes, investigation summaries, timelines, and case documentation.
All artifacts are sanitized and intended to demonstrate repeatable security operations workflows.

- SIEM Query Practice (Splunk)
- Packet Investigation (Wireshark)
- Case Documentation (TheHive)
